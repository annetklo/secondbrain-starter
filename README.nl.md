# Second Brain Starter Kit

Bouw een permanente AI-werkruimte in Claude Code met geheugen, vaardigheden en structuur die elke sessie slimmer wordt.

**Door [Mission Relearn](https://missionrelearn.com)**

## Snel Installeren

```bash
curl -sL https://lab.missionrelearn.com/package/install.sh | bash
```

Of bekijk het script eerst:

```bash
curl -sL https://lab.missionrelearn.com/package/install.sh -o install.sh
cat install.sh
bash install.sh
```

## Wat je krijgt

De starter kit installeert een compleet Second Brain fundament:

- **CLAUDE.md** - Je master-configuratie template
- **3 Kant-en-klare skills**:
  - `/meeting-prep` - Analyseer documenten en maak gestructureerde vergadervoorbereiding
  - `/morning-briefing` - Dagelijks agenda-overzicht met prioriteiten
  - `/skill-creator` - Bouw je eigen skills
- **Geheugensysteem** - Templates voor stem, stijl en voorkeuren
- **Kennisbank** - Merkrichtlijnen en referentie-templates
- **Hooks** - Automatisering bij sessie-start
- **Groeilog** - Volg de evolutie van je Second Brain

## Wat is een Second Brain?

Een Second Brain is een permanente laag van context, kennis en automatisering die Claude Code transformeert van een capabel hulpmiddel naar een gepersonaliseerde AI-partner. Eentje die je merkrichtlijnen kent, de juiste workflow op het juiste moment voorstelt, en elke sessie slimmer wordt.

Lees de volledige gids: [docs/guide.md](starter/docs/guide.md)

## Na Installatie

1. Open `~/.claude/CLAUDE.md` en vul je gegevens in
2. Draai `/morning-briefing` om je eerste skill te testen
3. Draai `/skill-creator` om een eigen skill te bouwen
4. Lees de [gids](starter/docs/guide.md) voor de volledige methodiek
