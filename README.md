# Second Brain Starter Kit

Build a persistent AI workspace in Claude Code with memory, skills, and structure that grows smarter every session.

**By [Mission Relearn](https://missionrelearn.com)**

## Quick Install

```bash
curl -sL https://lab.missionrelearn.com/install.sh | bash
```

Or if you prefer to inspect the script first:

```bash
curl -sL https://lab.missionrelearn.com/install.sh -o install.sh
cat install.sh
bash install.sh
```

## What You Get

The starter kit installs a complete Second Brain foundation:

- **CLAUDE.md** - Your master configuration template
- **3 Pre-built skills**:
  - `/meeting-prep` - Analyze documents and create structured meeting preparation
  - `/morning-briefing` - Daily calendar overview with priorities
  - `/skill-creator` - Build your own custom skills
- **Memory system** - Templates for voice, style, and preference tracking
- **Knowledge base** - Brand guidelines and reference templates
- **Hooks** - Session start automation
- **Growth log** - Track your Second Brain's evolution

## What Is a Second Brain?

A Second Brain is a persistent layer of context, knowledge, and automation that turns Claude Code from a capable tool into a personalized AI partner. One that knows your brand guidelines, suggests the right workflow at the right moment, and grows smarter with every session.

Read the full guide: [docs/guide.md](starter/docs/guide.md)

## The Six Layers

1. **Foundation** - CLAUDE.md + Memory
2. **Knowledge** - Brand, clients, tools, style
3. **Skills** - Automated workflows (3 included, 22+ available)
4. **Hooks** - Proactive automation
5. **Integrations** - MCP servers + specialized agents
6. **Health** - Quality tracking and maintenance

## After Installation

1. Open `~/.claude/CLAUDE.md` and fill in your details
2. Run `/morning-briefing` to test your first skill
3. Run `/skill-creator` to build a custom skill for your workflow
4. Read the [guide](starter/docs/guide.md) for the full methodology
