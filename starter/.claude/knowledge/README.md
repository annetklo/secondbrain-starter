# Knowledge Base

This directory contains deep reference material that Claude reads on demand (not every session).

## Suggested files to add:

- **brand.md** - Your visual identity, colors, fonts, document standards
- **clients.md** - Active clients, project context, key contacts
- **workflows.md** - Standard workflow chains and processes
- **tech-stack.md** - Tools, integrations, and system configurations

## How it works

Claude reads these files when they're relevant to your request. They don't consume context tokens until needed. Reference them from your CLAUDE.md so Claude knows they exist.
