# Example: Using a Skill

## Invoking a skill

Type the skill name as a slash command in Claude Code:

```
/meeting-prep
```

Claude will load the skill's instructions and guide you through the workflow.

## With arguments

```
/meeting-prep --person "Jane Smith, CTO, Acme Corp" --context "Partnership discussion"
```

## Creating your own skill

Use the skill-creator:

```
/skill-creator
```

Claude will walk you through:
1. Understanding what the skill should do
2. Planning the skill's resources
3. Initializing the directory structure
4. Writing the SKILL.md
5. Testing and iterating
